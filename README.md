# devhyun-page

Devhyun is a **personal branding** landing page

Website link: https://devhyun.com

## Project Stack

- NextJS
- React Query
- Emotion
- Axios
- Lodash
- Lottie Player

## Get started

If you already have Nodejs 16 and Yarn installed, type

```console
yarn install
yarn dev
```

to get started.
